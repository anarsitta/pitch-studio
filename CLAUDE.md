# Питч Студио — маркетинговый сайт

Одностраничный лендинг агентства полного цикла на Vue 3 + Vite. Подробности запуска и структуры — в `README.md`.

## Design Context

Перед любой работой над UI прочитай:

- **`PRODUCT.md`** — регистр (brand), аудитория, характер бренда (дерзкий, энергичный, современный), анти-референсы (шаблонные Tilda-лендинги, корпоративная сухость, инфоцыганщина) и дизайн-принципы. Главный: «сайт — это портфолио агентства».
- **`DESIGN.md`** — визуальная система «Неон на тёмной сцене»: сценический чёрный #0B0C0A + единственный акцент «кислотный лайм» #C4FA3B, Unbounded (дисплей) + Manrope (текст), глубина через лаймовый свет вместо серых теней. Токены в YAML-frontmatter нормативны.

Новые секции и правки должны следовать именованным правилам из DESIGN.md (одного прожектора, светящегося акцента, плотного дисплея, цветного света).
